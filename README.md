# 学词乐 (Xuecile) / Vocab Hello

这是一款主打“AI 定制词库”的外语学习工具，支持英语，法语，日语等主流外语。
拍个菜单、截个说明书，或者直接告诉 AI“我要开会用的英语动词”，它就能一键为你生成专属词库，通过 AI 对话辅导和拼写练习。搞定场景化词汇。拒绝无效词汇。

### ✨ 核心功能

- 🤖 **AI 辅导 (AI Chat)**：针对单个单词向 AI 提问（如用法、例句、词源）。
- ✍️ **拼写练习**：结合生成的专属词库进行针对性拼写测试，强化肌肉记忆。
- 🛠️ **词库生成器 (Vocabulary Generator)**：提供两种词库构建方式：
  - 💬 **大模型提取**：通过设定目标外语、生成数量和自定义需求描述（如“我是一名外企前端，给我一份日常开会汇报进度的动词”），AI 将自动生成带翻译和音标的专属词库。
  - 🖼️ **图片上传识别**：上传包含文字的图片（如课本、菜单或截图，试卷等），结合 AI 视觉识别技术，提取图片中的外语词汇并生成词库。
- 📖 **错词本 (Error Words)**：自动记录学习中出错的单词及错误次数，方便针对性复习。
- 🗂️ **词库管理**：支持多语言学习管理、自定义单词列表及进度追踪。
- ⚡ **技术栈**：前端使用 Vue 3 生态下的 **Nuxt 3** 和 **TailwindCSS**，后端基于 **Drizzle ORM** 和 MySQL 进行数据存储。

## 🐳 快速部署 (Docker 一键安装)

本项目推荐使用 Docker Compose 进行一键部署，内置了应用服务和 MySQL 数据库，开箱即用。

### 1. 环境准备
确保您的服务器或本地已安装 [Docker](https://docs.docker.com/get-docker/) 和 [Docker Compose](https://docs.docker.com/compose/install/)。

### 2. 配置 AI 模型 (环境变量)
复制 `.env.example` 为 `.env`，并填入您的 AI 模型配置。

```bash
cp .env.example .env
```


### 3. 一键启动
执行以下命令启动服务（首次启动会拉取镜像并初始化数据库，请耐心等待）：

```bash
docker-compose up -d
```

启动完成后，即可通过浏览器访问 `http://localhost:3000` 开始使用。
